<h2>📦 About</h2>
<p>
WebComponents is a collection of modern, reusable frontend code including UI elements,
authentication pages, layouts, and JavaScript utilities.  
Designed for developers who want clean, efficient, and scalable components.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
  <li>🎨 Ready-to-use UI components</li>
  <li>⚡ Lightweight and optimized code</li>
  <li>📱 Responsive and modern designs</li>
  <li>🔐 Authentication pages (Login / Signup)</li>
  <li>🧩 Modular JavaScript utilities</li>
  <li>🌐 Cross-browser compatible</li>
</ul>
